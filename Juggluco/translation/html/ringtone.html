Ringtone

Select which ringtone to play for an alarm or notification and for how many seconds.

Minutes deactivated: If a low or high glucose level continues, the alarm will not immediately go off again. You can specify after how many minutes the alarm should be generated again if the condition continues.

Sound: make sound.

Default: use default sound.

Select: Select a ringtone. On some Samsung phone the preinstalled ringtone picker app crashes, so you have to install a different ringtone app that can be used by other apps, for example: https://play.google.com/store/apps/details?id=com.centralparkapps.ringo.ringtones

Vibrate: Vibrate

Disturb: Also generate sound when "Do not Disturb" is on. If Juggluco doesn't have the required permission for this, you will be directed to an Android settings screen to give Juggluco this permission. By switching on "Do not Disturb" and pressing Play, you can test whether alarms will work during "Do not Disturb".

Flash: Smartphones with torch flashlight have the option to flash for the same number of seconds. Handy in case ringtones are not allowed and there is no clear policy concerning flashlights.

On sites like stackoverflow.com it is sometimes claimed that some smartphones crash when using Flash light without camera permission, if that is the case you can turn on Camera permission.
