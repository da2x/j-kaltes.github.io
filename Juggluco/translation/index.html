Translate Juggluco


To translate Juggluco the following files have to be translated:

  • strings.xml containing the labels used by the Android interface:

    • German translation;

    • Italian translation;

    • Dutch translation.

    • Portuguese translation.

  • jugglucotext.cpp containing the text used in the CXX part of Juggluco:

    • German translation;

    • Italian translation;

    • Dutch translation.

    • Portuguese translation.

  • The help texts are written in Libreoffice and saved in html format: Separate, Zip;

    • German translation;

    • Italian translation;

    • Dutch translation.

    • Portuguese translation.

In the help texts, the terms in strings.xml and jugglucotext.cpp are used, so you can better translate the terms first and the help texts later.

The filenames of the help texts should stay the same, but placed in a different directory, named for example fr for French.

The translation of jugglucotext.cpp can be saved in frjugglucotext.cpp.
